# JumpApp

**JumpApp** 是一款极简的跨平台聚合搜索与跳转工具（Android App）。
支持输入关键字后，一键快速跳转到各大热门平台的原生 App 中进行搜索。

## 支持平台
- 哔哩哔哩 (Bilibili)
- 抖音 (Douyin)
- 微博 (Weibo)
- 小红书 (Xiaohongshu)
- 知乎 (Zhihu)

## 技术栈
- 100% Kotlin
- Jetpack Compose
- Kotlin DSL + Gradle

## 运行与编译
1. 使用 **Android Studio** 打开本工程。
2. 同步 Gradle 依赖。
3. 连接您的 Android 设备或模拟器，点击运行即可。
